## 当前状态

硬件已经完成样机和上位机联调，第一版可读原理图已经公开：

- [`sbus-rs422-bridge-schematic.pdf`](sbus-rs422-bridge-schematic.pdf)：单页 A4 横版原理图，包含供电、SBUS 输入与反相、隔离 RS-422 收发、模式选择和差分接口。

可编辑原理图源文件、PCB、BOM、制造文件及版本标识仍在整理中，因此当前提交用于设计理解与评审，不构成完整的可生产硬件包。

请不要根据本页的功能描述自行生产或接入高风险控制系统。

## 已验证的设计目标

- 接收标准反向电平 SBUS，并通过当前启用的反向器转换为非反向内部 UART 逻辑；
- 保持 `100000 bit/s / 8E2`、帧率和 25 字节数据内容不变；
- 输出 RS-422 互补差分信号；
- SBUS 输入地与 RS-422 输出地电气隔离；
- 支持通过 USB→RS-422 适配器接入本仓库上位机；
- 已使用标准 SBUS 信号源完成连续收发、帧解析和输入/输出一致性验证。

## 原理图功能说明

### 电源与隔离

- 通过 Type-C 或两针接口输入 5 V；
- 低压差稳压器产生 3.3 V 逻辑电源；
- 隔离式 RS-422 收发器把 SBUS 侧逻辑地与 `GND_RS422` 分开；
- 差分接口包含 120 Ω 端接与保护器件。

### 可选择的逻辑反向器

标准 SBUS 在线路侧使用反向 UART 电平，而常规 USB 串口解码器通常接收非反向 UART 逻辑。原理图在接收和发送方向各放置一路可选反向器，使同一硬件能够根据外部设备极性选择“反向转换”或“旁路直通”。

| 路径 | 选择元件 | 原理图当前装配 | 当前作用 |
| --- | --- | --- | --- |
| SBUS 接收 | `R3 / R15` + `U3` | `R3 = 0 Ω`、`R15 = NC` | 启用反向器：标准反向 SBUS → 非反向内部逻辑 |
| SBUS 发送 | `R10 / R16` + `U5` | `R10 = 0 Ω`、`R16 = NC` | 启用反向器：内部逻辑 → 反向 SBUS 输出极性 |

需要接入已经是非反向 UART 极性的信号时，可以通过互换相应的 `0 Ω / NC` 装配选项选择旁路路径。修改装配前应结合可编辑原理图源文件和实物版本复核，不能仅凭 PDF 丝印直接改焊。

这里的“反向”只描述逻辑电平极性。反向器不会有意修改 UART 波特率、SBUS 帧率、25 字节帧结构或通道数据。当前项目的实际使用方式是：外部输入标准反向 SBUS，启用接收反向器，再经隔离 RS-422 输出到 USB→RS-422 适配器。

### 收发模式选择

原理图通过 `DE / RE` 控制隔离收发器状态：

| DE | RE | 模式 |
| --- | --- | --- |
| 0 | 0 | 接收模式 |
| 0 | 1 | 浮空模式 |
| 1 | 0 | 双向模式 |
| 1 | 1 | 发送模式 |

当前仓库完成验证的是标准反向 SBUS 输入经反向器转换后输出 RS-422 的链路。发送方向和双向模式是硬件预留能力，尚未纳入当前软件与测试报告的验收结论。
